## Features
* Recently failed words have higher probability of showing.
* Daily stats for successful and wrong answers. 

## Yandex API key
Open encrypted credentials:
```
rails credentials:edit
```

Add Yandex API key:
```
yandex:
  api_key: <your_key_here>
```
